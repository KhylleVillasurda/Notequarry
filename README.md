# NoteQuarry

**Version:** 1.0  
**Author:** Khylle P. Villasurda

## Overview

NoteQuarry is a secure, high-performance desktop note-taking app built with **Rust and Qt**. It supports two modes:
- **Book Mode:** Paginated notes with word count tracking
- **Note Mode:** Freeform notes with checkboxes and rich text

### Features

- **End-to-end encryption** (ChaCha20-Poly1305, Argon2id)
- **Cloud sync** (Google Drive, Proton Drive coming soon)
- **Fast search** across all entries
- **Dark theme UI** with custom styling
- Handles 10,000+ entries efficiently

### Tech Stack

- **Frontend:** Qt 6.10.0 (C++17) - Cross-platform desktop UI
- **Backend:** Rust - Core logic and security
- **Database:** SQLite - Local storage
- **Cloud:** Google Drive API (Proton Drive soon)
